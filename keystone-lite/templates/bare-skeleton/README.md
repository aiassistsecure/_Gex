# Bare Skeleton

Minimal project structure for AI-assisted development.
This is a blank canvas - describe what you want to build and let AI help you create it.

## Getting Started

1. Open this folder in Keystone Lite
2. Describe what you want to build in the chat
3. Let AI create the files and structure for you

## Suggested Prompts

- "Create a simple REST API with Node.js"
- "Build a React component for a todo list"
- "Set up a Python CLI tool"
- "Create a web scraper"
